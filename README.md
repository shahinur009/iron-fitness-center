## ğŸ“¦ Installation

Follow these steps to clone and run the project locally:

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/iron-fitness.git
   cd iron-fitness
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
